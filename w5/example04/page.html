Drawn Paths

This example sets up a field layer & a test layer. It calls for a specific Function Set from the Helpers class and passes the test layer to be used for drawing var fnSet = Helpers.getFnSet("drawPath", {layerToUse: testLayer});.

To Draw Paths:

  1. Click (onMouseDown): start new path
  2. Drag (onMouseDrag): adds points
  3. Release (onMouseUp): smooths/simplifies the path and draws a calculated number of performers
